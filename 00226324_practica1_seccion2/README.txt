Ejercicio 2. Utilizando etiquetas para texto

Hay cambios en la visualización del sitio web? 
RESPUESTA: No, no hay cambios visibles porque las etiquetas <meta> no se muestran en el navegador

 ¿Para qué sirven las etiquetas meta?
 RESPUESTA: Sirven para proporcionar información adicional sobre la página (autor, descripción, palabras clave, codificación, etc.). Aunque no cambian la apariencia, son importantes para la accesibilidad, y la correcta interpretación de la página por navegadores y motores de búsqueda.


Ejercicio 3. Agregando vínculos e imágenes

¿Qué pasaría si la imagen esta guardada
en la misma carpeta de la página web? 
RESPUESTA: El valor de src sería simplemente "imagen.jpg" sin necesidad de poner la ruta de carpeta, ejemplo: <img src="imagen.jpg" alt="Imagen">


¿Y si está en una carpeta superior?
RESPUESTA: Se debe usar ../imagen.jpg para indicar que la imagen está un nivel arriba en la estructura de carpetas, ejemplo: <img src="../imagen.jpg" alt="Imagen">


 ¿Cómo se consigue que el último enlace se abra en una nueva ventana?
 RESPUESTA: Se logra con el atributo target="_blank" dentro de la etiqueta <a>
